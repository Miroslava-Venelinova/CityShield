# CityShield

# What is CityShield?
CityShield is a mobile application that provides up-to-date information about power, water, and other utility outages in urban areas in Varna, Bulgaria.

- Users receive real-time notifications about any interruptions in essential infrastructure, based on official sources such as energy and water supply companies, local authorities, and other trusted institutions. 
This way, users will be informed in a timely manner about possible inconveniences, allowing them to plan their daily lives more effectively and prepare for situations such as accidents or repairs. 

# CityShield guarantees the reliability and accuracy of the information by providing up-to-date data directly from official channels to ensure maximum transparency and comfort for consumers.
